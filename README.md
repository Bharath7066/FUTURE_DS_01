# FUTURE_DS_01
Data Science & Analytics Internship Task 1 – Business Sales Performance Analysis using Python, Excel and Power BI

📊 Business Sales Analysis Project
📌 Objective
Analyze sales data to help a business understand performance and improve growth.

🔍 Key Questions & Answers
1. Which products generate the most revenue?
The top-performing products are those in the Technology category, contributing the highest sales.

2. How do sales change over time?
Sales show an increasing trend over time, indicating business growth.

3. Which region performs best?
The West region generates the highest revenue compared to others.

4. Where should the business focus?
The business should focus on high-performing products and expand in top regions.

📈 Tools Used
Python (Google Colab)
Pandas
Matplotlib
💡 Key Insights
Technology category dominates revenue
Some regions underperform and need improvement
🚀 Recommendations
Increase marketing in top regions
Focus on best-selling products
Improve low-performing categories

# Superstore Dashboard

This Power BI Superstore Dashboard project provides an interactive analysis of sales, profit, orders, and customer performance using retail business data. The dashboard includes visualizations such as sales trends, profit by category, regional performance, top-selling products, and customer insights. KPI cards and charts were used to monitor overall business growth and identify high-performing segments. Data cleaning and transformation were completed using Power Query to improve data quality and accuracy. Interactive slicers and filters help users explore data dynamically for better decision-making. The project demonstrates business intelligence and data visualization skills using Power BI to generate meaningful insights and support strategic planning.
